# ML-projects

Studied two datasets of numerical and categorical type and analysed their purpose, source, features, number of classes, and performance achieved by various authors.  
Extracted and stored the two datasets using python.
Implemented Train–Test–Validation split on a dataset and tuned hyperparameters using a learning model. 
Implemented different training methods such as Holdout Method, K-Fold Validation, Stratified Random Sampling, and Bootstrap Sampling and compare their performance using a learning model.
Implemented a Bayesian classification model with multiple features, multiple values, and multiple classes in the NLP domain by performing text cleaning, stemming, lemmatization and handling borderline classification cases.
Implemented the k-Nearest Neighbors algorithm using normalization , Linear Regression using the Gradient Descent algorithm and display the convergence plot.
Implemented regularization, Feature selection using Variance Threshold method and Pearson Correlation method
